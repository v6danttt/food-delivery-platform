# Food Delivery Platform Backend

## Overview

This repository contains the **backend** for a food‑delivery platform. It provides APIs for managing restaurants, menus, orders, and delivery logistics. The codebase is written in **Node.js** with Express, and uses a MongoDB database (or whichever is configured in the `.env`).

## Getting Started

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/v6danttt/food-delivery-platform.git
   cd food-delivery-platform
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure environment variables**:
   - Copy the example file:
     ```bash
     cp .env.example .env
     ```
   - Edit `.env` and set the required values (e.g., `PORT`, `MONGODB_URI`, JWT secrets, etc.).
4. **Run the server**:
   ```bash
   npm start   # or `npm run dev` for hot‑reloading with nodemon if configured
   ```
   The API will be available at `http://localhost:<PORT>`.

## API Documentation
